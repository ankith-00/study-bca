Computer Architecture



NUMBER SYSTEM

Computer architecture can be defined as a set of rules and methods that describe the functionality, management and implementation of computers.

The number system is a way of expressing numbers using symbols which serve as mathematical language.



TYPES

  • Binary number system

    The binary number system uses 2 digits 0 and 1 represented by base 2.
    Example : (101001) 2

  • Decimal number system

    The decimal number system uses 10 digits starting from 0 - 9. Represented by base 10.
    Exampl : (198) 10

  • Octal number system

    The octal number system uses 8 digits from 0 - 7 , represented using base 8.
    Example : (47) 8

  • Hexa-decimal number system

    The hexadecimal number system uses 16 symbols combining 0 - 9 digits and A - F alphabets, represented by base 16.
    Example : (A47E) 16
